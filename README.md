# 약국 추천 서비스
Java 와 Spring Boot 를 기반으로 kakao API 를 사용하여 가까운 약국에 대한 정보와
길찾기, 로드뷰 기능을 제공합니다.

## 개발 환경
* Intellij IDEA Ultimate
* Java 11
* Gradle 7.4.1
* Spring Boot 2.6.7

## 기술 세부 스택
* Spring Web
* Spring Data JPA
* Rest Repositories
* Redis Database
* MariaDB Driver
* Lombok
* handlebars
* Spring Boot DevTools
* Spring Configuration Processor
* spock
* testcontainers
* spring retry

그 외
* AWS EC2
